Seven Segment Display with 8 Bit Shift Register

github link

This small project is simply a seven segment display that displays an integer between 0 and 255.
The value is adjusted with the rotary encoder (knob) on the right. Pressing the knob resets the value to 0, and if the value is already 0 it gets set to 255

It functions by sending 8 bits serially to a shift register. The first four bits are then sent to a 7447 BCD decoder to determine what integer to show, and the second four bits are sent to the pins of the SSD itself to determine which of the digits to show.

The seven segment display itself functions by telling it one number, and one of the four available digits to display that number on, and then rapidly switching between digits to give the illusion of displaying numbers larger than 9.

Piezo Piano

github link

This project is 3 buttons used as the "keys" for the piano, a piezo buzzer to generate the sound, and a potentiometer that controls the octaves played.
I would have liked to have had 12 buttons to fully emulate one section of a piano, but unfortunately I only had 3 buttons in my box of parts available to me at the time.
To make up for the lack of notes to play, I added the potentiometer which controls the octave of the notes being played.

The notes playable are, in order: C, D, and E. The specific frequencies can be found in the arduino .ino file.
The potentiometer allows for three ranges of octaves that the user can change between.

Relation Database System Normalizer

github link

This project was my final submission for my 5000 level database systems course from my last semester of college. The program functions by taking in a table as a .csv file, and will then determine which normal form that table already exists in. The user can then input which normal form, up to Boyce-Codd Normal Form, they would like to translate the table into.
